# Jingchao Wang Homepage

Static personal homepage inspired by the structure of `https://luliangsi.github.io/`, rebuilt with a custom visual design and content.

## Files

- `index.html`: page structure
- `styles.css`: styles and responsive layout
- `script.js`: publication data, interaction chips, mobile menu, reveal animation
- `assets/avatar.jpg`: profile image

## Local preview

```bash
python -m http.server 8000
```

Then open `http://localhost:8000` in a browser from this directory.

## Deploy to GitHub Pages

Push these files to the root of a GitHub repository named `<username>.github.io`, or enable GitHub Pages for this folder in any repository.
